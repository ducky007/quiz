-------------------------------------------------------------------------------
-- quiz --
-------------------------------------------------------------------------------

This is the software developed to drive a quiz buzzer system. The electronic is based on a Arduino Uno and an Adafruit Wave Shield. The system supports 8 players, one reset button, two high current LEDs and 8 LEDs driven by a 74HC595 multiplexer. 

You will find the latest version of this code at the following address:
http://github.com/pchretien
